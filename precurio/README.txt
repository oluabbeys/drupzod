Precurio v2.1 Readme
---------------------

Thank you for downloading Precurio!

In a few words, Precuro is an easy to use, visually attractive, enterprise 2.0 intranet portal software with workflow automation and robust reporting. 

Precurio is offered as an open source software. Download it for free, customize it to your needs, add new functionalites and share them with the community. Although distributed as free software, Precurio is being supported by a professional team of highly skilled developers.

If you are upgrading your Precurio installation, please read INSTALL.TXT.
If this is a new installation, please read UPGRADE.TXT
